# eslint-config-next
ESLint rules for Next.js

## Installation

1. Run the install command
   ```bash
   $ npm install -D @bilbase-inc/eslint-config-next
   ```
2. You also need to install peer dependencies
   ```bash
   $ npm install -D eslint
   ```

## Usage

`.eslintrc.yaml`

```yaml
extends:
  - "@bilbase-inc/eslint-config-next"
```
